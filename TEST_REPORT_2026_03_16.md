## ⚠️ Найденные проблемы

### 1. Пустые ответы API

**Проблема:** Некоторые endpoints возвращают пустые ответы (`200 OK` без тела)

**Endpoints:**
- `GET /api/project/2/inventories` — пусто (ожидаются инвентари)
- `GET /api/project/2/tasks/last` — пусто (ожидаются последние задачи)

**Возможная причина:**
- В проекте отсутствуют данные инвентарей/задач
- Endpoint требует другого формата URL

**Решение:**
- Проверить наличие данных в БД
- Свериться с документацией API

---

### 2. Сервер останавливается

**Проблема:** Сервер неожиданно остановился после тестов

**Возможная причина:**
- Завершение процесса после запросов
- Паника в коде (не зафиксирована в логе)

**Решение:**
- Добавить обработку паник
- Проверить логи при следующих запусках
