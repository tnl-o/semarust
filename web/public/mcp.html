MCP / AI Gateway

Model Context Protocol

Встроен в Velum · Не требует отдельного сервиса · MIT

Аутентификация

MCP использует тот же JWT-токен, что и REST API Velum.
Получите токен в Профиле → API Токены.

Храните токен в безопасном месте. Он даёт полный доступ ко всем проектам пользователя.
