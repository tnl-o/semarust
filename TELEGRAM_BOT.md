## 📖 Обзор

Telegram бот предоставляет удобный способ взаимодействия с Velum через мессенджер Telegram.

**Возможности:**
- ✅ Просмотр статуса системы
- ✅ Просмотр проектов
- ✅ Просмотр последних задач
- 🔜 Уведомления о завершении задач
- 🔜 Интерактивное управление задачами

## 🚀 Настройка

### 1. Создание бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Введите имя бота (например: `Velum Bot`)
4. Введите username бота (например: `semaphore_example_bot`)
5. Сохраните полученный токен

### 2. Настройка Semaphore

Добавьте токен бота в конфигурацию:

**Вариант 1: Переменная окружения**
```bash
export SEMAPHORE_TELEGRAM_BOT_TOKEN="1234567890:ABCdefGHIjklMNOpqrsTUVwxyz"
```

**Вариант 2: Конфигурационный файл**
```json
{
  "telegramBotToken": "1234567890:ABCdefGHIjklMNOpqrsTUVwxyz"
}
```

**Вариант 3: .env файл**
```bash
SEMAPHORE_TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

### 3. Запуск

```bash
# Запуск сервера
cargo run -- server --host 0.0.0.0 --port 3000

# Бот автоматически запустится если токен настроен
```

## 💬 Команды

### `/start` - Запустить бота

Запускает бота и показывает приветственное сообщение.

**Пример:**
```
👋 Добро пожаловать в Velum Bot!

Я помогу вам управлять задачами и проектами.

Доступные команды:
/help - Показать справку
```

---

### `/help` - Показать справку

Показывает список доступных команд и примеры использования.

**Пример:**
```
📖 Справка по командам

/start - Запустить бота
/help - Показать эту справку

План развития:
- /status - Статус системы
- /projects - Список проектов
- /tasks - Последние задачи
```

---

### `/status` - Статус системы (в разработке)

Показывает общую статистику системы.

**Планируемый вывод:**
```
📊 Статус Velum

Пользователей: 10
Проектов: 5
Активных задач: 3
Время: 2026-03-10 15:30:00
```

---

### `/projects` - Список проектов (в разработке)

Показывает список всех проектов.

**Планируемый вывод:**
```
📁 Проекты:

1. Production (ID: 1)
2. Development (ID: 2)
3. Staging (ID: 3)
```

---

### `/tasks [limit]` - Последние задачи (в разработке)

Показывает последние задачи.

**Примеры:**
```
/tasks      - Последние 5 задач
/tasks 10   - Последние 10 задач
```

**Планируемый вывод:**
```
📋 Последние задачи:

Production:
✅ Задача #123 (Шаблон: Deploy)
❌ Задача #122 (Шаблон: Build)

Development:
⏳ Задача #121 (Шаблон: Test)
```

## 🔔 Уведомления

### Настройка уведомлений

Для получения уведомлений о задачах:

1. Отправьте боту команду `/start`
2. Запомните ваш Chat ID (можно узнать у бота @userinfobot)
3. Добавьте Chat ID в настройки профиля Semaphore

### Типы уведомлений

| Событие | Эмодзи | Описание |
|---------|--------|----------|
| Задача успешна | ✅ | Задача выполнена успешно |
| Задача провалена | ❌ | Задача завершилась ошибкой |
| Задача запущена | ⏳ | Задача начала выполнение |
| Задача остановлена | ⏹️ | Задача была остановлена |

### Пример уведомления

```
✅ Задача: #123

Проект: Production
Шаблон: Deploy
Статус: success
Время: 2026-03-10 15:30:00
```

## 🔐 Безопасность

### Рекомендации

1. **Храните токен в секрете**
   - Не коммитьте токен в Git
   - Используйте переменные окружения
   - Ограничьте доступ к конфигурации

2. **Используйте whitelist Chat ID**
   - Разрешите только авторизованным пользователям
   - Настройте фильтрацию по ID

3. **Ограничьте команды**
   - Отключите опасные команды
   - Настройте права доступа

## 🛠️ Разработка

### Добавление новой команды

1. Откройте `src/services/telegram_bot/mod.rs`
2. Добавьте обработчик команды:

```rust
fn my_command_handler(&self) -> dptree::Handler<'_, teloxide::RequestError> {
    use dptree::case;
    
    case![BotCommand::MyCommand].endpoint(|bot: Bot, msg: Message| async move {
        bot.send_message(msg.chat.id, "Hello!").await?;
        Ok(())
    })
}
```

3. Добавьте команду в enum `BotCommand`
4. Зарегистрируйте обработчик в `run()`

### Тестирование

```bash
# Запуск с тестовым токеном
export SEMAPHORE_TELEGRAM_BOT_TOKEN="test_token"
cargo run -- server
```
